# Feature Research - Part 1 (Core MVP Features)

## Must-Have Features (MVP)
1. **Unstructured Text Ingestion**: Ability to paste or type raw notes quickly without needing tags.
2. **Local AI Parsing**: A lightweight local model to extract key entities (Dates, Topics, Tasks) from raw text.
3. **Semantic Search**: Ability to find notes based on meaning, not just exact keywords.
4. **SQLite Storage Backend**: Durable, relational local storage for all memories.
5. **Basic Reminder System**: Time-based triggers for extracted tasks.

## Why MVP?
These form the absolute core loop: Input -> Process -> Store -> Recall. Without these working smoothly, no advanced features matter.
